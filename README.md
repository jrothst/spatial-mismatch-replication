# Replication Package: Reassessing the Spatial Mismatch Hypothesis

**Authors:** David Card, Jesse Rothstein, and Moises Yi
**Publication:** AEA Papers and Proceedings, 2024

## About This Repository

This repository contains the replication package for "Reassessing the Spatial Mismatch Hypothesis" (Card, Rothstein, and Yi, 2024).

## Official Repository

The official published replication package is maintained on openICPSR:
- https://www.openicpsr.org/openicpsr/project/200881/version/V1/view

This GitHub repository is a mirror for convenience and version control purposes.

Note that the replication package is for the AEA-PP paper. There is also an extended working paper with additional results; this package produces most but not all of these results.

## Documentation

**For complete replication instructions, data documentation, and details, please see the `README.pdf` file in this repository.**
